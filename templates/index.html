About This System

This system uses Natural Language Processing (NLP) and machine learning to detect fraudulent mobile money SMS messages targeting Ghanaian users. The pipeline implements TF-IDF vectorisation (5,000 features, unigrams + bigrams), text preprocessing (lowercase, punctuation removal, stop-word removal, lemmatisation), and a Logistic Regression classifier achieving 96.67% accuracy on 1,200 Ghanaian MoMo SMS messages. Built with Python, NLTK, Scikit-learn and Flask.
