How It Works
🧹

Text Preprocessing

Lowercasing, punctuation removal, stop-word filtering and lemmatisation clean each message before analysis.

📊

TF-IDF Vectorisation

5,000-feature TF-IDF with unigrams and bigrams converts text into numerical representations the model understands.

🤖

Logistic Regression

The best-performing classifier among Naive Bayes, LR and SVM — achieving 96.67% accuracy on Ghanaian MoMo data.
